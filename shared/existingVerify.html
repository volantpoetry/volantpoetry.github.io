🔧 Update Verified Badges

This tool will add isVerified: true to all users who have already verified their email.

⚠️ Warning: This should only be run once to update existing users.
New users will automatically get the field when they verify.
ℹ️ What this does:
- Scans all users in Firestore
- Checks if they have emailVerified: true
- Adds isVerified: true to their profile
- This enables the verified badge to appear
